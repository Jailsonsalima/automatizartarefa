# 📦 PowerStock

Um sistema simples de **cadastro de produtos** desenvolvido em **Django**, com autenticação básica e persistência de dados.  
O fluxo é bem direto:  
- O usuário faz login com qualquer e-mail e senha (a conta é criada automaticamente).  
- Após o login, é exibido um formulário para cadastrar produtos com informações como código, marca, tipo, categoria, preço e custo.  
- Os produtos cadastrados ficam vinculados ao login e aparecem listados abaixo do formulário.  
- Se o mesmo usuário acessar novamente, os dados dos produtos cadastrados anteriormente são exibidos.  
- Há também a opção de limpar todos os produtos cadastrados.  

### 🚀 Tecnologias utilizadas
- Python 3  
- Django  
- HTML + CSS (responsivo)  
- SQLite (banco de dados padrão do Django)  

### 🎯 Objetivo
Facilitar o gerenciamento de produtos de forma prática e rápida, servindo como base para estudos de **python, pyautogui e automatização de tarefas**.
